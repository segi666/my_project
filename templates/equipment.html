등산화 선택하기
1. 5~10mm 정도 큰 것 구입
소비자들이 등산화에 불편을 가장 많이 호소하는 것은 발가락이 앞부리에 닿아 고통을 호소하는 경우와 발바닥의 통증, 무릎 통증, 등산화 무게와 부드러움에 대한 불평 등이다.
2. 엄지발가락 부위에서 꺽이는 것을 구입
구입시 신발을 신고 뒷꿈치를 들어 엄지발가락 부위에서 등산화가 꺽이는 것을 선택하며 이때 발등부위를 누르지 않는지 확인한다. 발등이 구부러지면 발이 아파 장시간 워킹을 불가능하게 한다.
3. 창이 너무 단단하지 않은 것을 고른다
창이 부드러운 것은 미끄러지지 않는 반면 단단한 것은 내마모성은 강하나 미끄러지기 쉽다. 특히 서울의 산은 바위로 이뤄져 있기 때문에 부드러운 창을 선택한다.
4. 창의 두께는 두툼한 것이 좋다
창의 두께는 산행의 피로를 줄이는데 중요한 역할을 한다. 너무 얇으면 쉬이 피로해지고 장시간 걸으면 무릎에 무리가 온다. 구입시 신어 보아 탄력이 없으면 중창이나 깔창을 깔아 두께를 보완 한 후 사용하면 좋다.

등산복
1. 등산복 고르는 법
겨우 우리나라 산 타면서 무슨 등산복을 입어? 라고 얘기하는 이들이 있다. 하지만 우리나라의 산은 계절별 기온 차이와 강수량 차이가 매우 크고 기상 변화가 잦다. 때문에, 등산복은 사치가 아닌 꼭 필요한 장비이다. 아래 기준으로 선택해보자!

  • 땀이 빠르게 마르는 소재
  • 행동하는데 불편이 없는 옷
  • 천이 질기고 튼튼한 옷
  • 가볍고 보온이 잘 되는 옷
  • 방수성과 통기성이 좋은 옷
    • 계절별 등산복 입는 방법
    2. 겹쳐 입기(레이어링)

    속옷, 보온옷, 겉옷 세가지 종류의 등산복을 겹쳐 있는 것을 레이어링이라고 한다.

  • 속옷 : 촉감 좋고, 땀이 잘마르는 기능성 옷
      ex) 베이스레이어
  • 보온옷 : 보온기능은 뛰어나지만 비바람에는 약한 옷
      ex) 플리스재킷, 다운재킷
  • 겉옷 : 비·바람을 막아주고, 입고 벗기가 편한 옷옷
      ex) 바람막이
    • 등산 배낭
    1. 등산 배낭 고르는 법
    △ 산행 목적에 따른 배낭 종류

    20-25L 배낭은 당일 산행용으로 적합하긴 하지만, 배낭이 너무 작으면 짐을 하나 뺄 때마다 내용물을 다 꺼내야 하는 등 불편이 많다. 또 산행 횟수가 늘어나면 처음 생각과 달리 내용물이 늘어난다. 따라서 30-40리터 배낭을 추천한다.

  • 토르소(목부터 골반까지) 길이에 맞출 것
  • 목은 고개를 앞으로 숙였을 때 톡 튀어나는 뼈
  • 골반은 등을 만졌을 때 골반과 척추 교차지점
  • S:41~47cm, M:45~52cm, L:51~57cm
    • 2. 등산 배낭 사용법
    △ 배낭을 맬 때는 ⓘ먼저 허리벨트를 조이고, ②어깨끈을 당기고, ③상단의 조절끈을 당겨 등에 배낭을 밀착시키고, ④마지막으로 가슴벨트를 연결한다. ⑤ 다리를 90도로 올렸을 때 허리벨트의 하단과 수평이 되어야 올바르게 착용한 것!

  • 조임 끈을 다 푼 상태에서 배낭을 꾸릴 것
  • 가벼운 것을 아래로, 무거운 짐을 위에 넣을 것
  • 부드러운 물건을 등쪽에 넣어 피로를 줄일 것
  • 무게가 대칭이 되도록 할 것
  • 산행 중 바로 꺼내 쓰는 장비는 배낭헤드에 넣을 것
  • 옷/식량은 비닐 등에 담아 분리해서 보관할 것
  • 배낭지퍼를 10시나 2시 방향으로 닫을 것
    • 준비물
    1. 겨울 산행 준비물
  • 등산화 : 산행 전 왁스를 발라 눈밭에 젖지 않게 함
  • 배낭 : 당일산행 - 35L이상, 1박이상 - 80L이상
  • 스패츠 : 방수투습성 원단, 당김끈 장식 내구성 확인
  • 아이젠 : 체인수 많은 것, 벨크로 달려있는게 편함
  • 등산스틱 : 접이식이 휴대 편함
  • 보온병 : 혹한에 온도 유지에 좋음
  • 물 : 넉넉히 준비, 조금씩 천천히 마실 것
  • 행동식 : 양갱, 단팥빵, 초코바, 육포 등 추천
  • 다용도칼
  • 선글라스, 고글 : 눈에 비친 반사광 차단
  • 시계 : 방수, 야광 필수, 고도계 겸용이면 더욱 좋음
  • 내의 : 면직은 절대 금물, 기능성 베이스레이어 추천
  • 양말 : 모직양말 2켤레 이상 필수
  • 모자 : 귀가 눈을 덮을 수 있는 발라클라바 형태 추천
  • 장갑 : 기능별 3켤레 준비(얇은, 두터운, 오버글러브)
  • 바지 : 모직바지 또는 두터운 합성섬유 원단 바지 추천
  • 바람막이 : 방수투습성 원단 추천
  • 우모복 : 후리스, 다운패딩류 추천
    • 2. 야영 시 추가 준비물
  • 침낭 : 최소 1kg 이상 다운함량 들어있는 것 준비
  • 에어매트 : 동계용, 키보다 조금 긴 것으로 구입
  • 텐트 : 양방이 트이지 않은 것, 플라이에 밑단을 낸 것
  • 헤드랜턴 : 산행전 작동여부 확인 필수
  • 코펠 : 산행 인원수에 맞게 챙길 것
